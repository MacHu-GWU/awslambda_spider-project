Install
------------------------------------------------------------------------------

``awslambda_spider`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install awslambda_spider

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade awslambda_spider
